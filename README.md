\# 🌌 N.I.N.A. AI Cortex



\*\*Cognitive Overlay \& Multi-Agent Supervisor\*\* для астрофотографического комплекса N.I.N.A.



\## 🚀 Возможности

\- \*\*Read-Only Ingestion:\*\* Мониторинг телеметрии, логов и метаданных без вмешательства в работу N.I.N.A.

\- \*\*Shadow Sequencer:\*\* Теневой граф секвенсора с предиктивной аналитикой.

\- \*\*Safe Injection:\*\* Безопасное изменение глобальных переменных и эмуляция триггеров через Advanced API.

\- \*\*Multi-Agent Swarm:\*\* AI-агенты (Стратег, Аналитик, Хранитель) для принятия автономных решений.



\## 🛠 Стек

\- \*\*Backend:\*\* Python 3.11+, FastAPI, LangGraph, Pydantic v2

\- \*\*Data:\*\* InfluxDB, DuckDB, Redis, Qdrant (RAG)

\- \*\*Frontend:\*\* Vue 3, Nuxt, Apache ECharts
